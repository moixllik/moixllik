# Interfaz de usuario

> Interfaz de usuario basado en 'Todo lo que necesitas lo tienes desde el inicio'

## `&lt;Mod&gt;` Buscar entre el contenido

<img alt="ui-search" src="articles/es-idea-ui/ui-search.svg" style="float:right;margin-left:1em;height:240px">

De la forma más rápida para realizar una búsqueda entre el contenido en la posición actual.

- `@KeyPress`.- Barra de entrada de texto en la parte superior.
- `&lt;Esc&gt;`.- De naranjado el origen o inicio.
- `&lt;Arrow&gt;`.- De amarillo las alternativas de selección.
- `&lt;Mod + Arrows&gt;`.- De gris las siguientes alternativas.
